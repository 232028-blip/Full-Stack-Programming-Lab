# Assignment 02 - Next.js Conversion

This project converts Assignment 01 (static HTML) into a Next.js app using the App Router, TypeScript, and Tailwind CSS. All pages are preserved as routes and shared layout components are used for the header, navigation, and footer.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Local Development

- Install dependencies: `npm install`
- Start the dev server: `npm run dev`
- Production build: `npm run build`
- Run production server: `npm run start`
